# Visuals Fallback Placeholder

This file should be replaced with an actual WebP image that will be used as a fallback when visual images fail to load.

## Specifications
- **Format**: WebP
- **Dimensions**: 800x1200px (or maintain 2:3 aspect ratio)
- **File Size**: Under 100KB
- **Content**: A neutral placeholder image (e.g., gradient, pattern, or simple geometric shape)

## Usage
This image will automatically be used when:
- A visual work image fails to load
- An image URL is broken or inaccessible
- An image hasn't been uploaded yet

## How to Add
1. Create or find a suitable placeholder image
2. Convert to WebP format
3. Name it exactly: `visuals-fallback.webp`
4. Place it in: `/public/images/placeholders/`
