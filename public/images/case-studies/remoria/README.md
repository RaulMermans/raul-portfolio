## 🔄 After Uploading

Once you upload images with the correct filenames, the code will automatically use them instead of placeholders.

**No code changes needed!** Just upload the images with the exact filenames above.

---

## 📝 Image Format

- **Format:** WebP (recommended) or JPG/PNG
- **Optimization:** Compress images before uploading for better performance
- **Naming:** Use lowercase with hyphens (e.g., `hero.webp`, not `Hero.webp`)
