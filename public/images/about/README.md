# About Section Images

This folder contains images for the About section and About page.

## Current Images

- `profile.webp` - Profile photo of Raúl Mermans (used in About component and About page)

## How to Add/Replace Images

1. **Save your image** to this folder (`/public/images/about/`)
2. **Recommended format**: WebP for best performance
3. **Recommended size**: 800x800px or larger (square aspect ratio preferred)
4. **Update the path** in:
   - `components/About.tsx` (homepage About section)
   - `app/about/page.tsx` (About page)

## Image Requirements

- Format: WebP (preferred), JPG, or PNG
- Size: At least 800px wide/height
- Aspect ratio: Square (1:1) preferred
- File size: Keep under 300KB if possible (Next.js will optimize)
