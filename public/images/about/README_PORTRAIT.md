# Portrait Image - Quick Reference

## Where to Save

Save your portrait image as:
```
profile.webp
```
in this folder: `/public/images/about/`

## Full Path
```
/Users/raulm/Desktop/GitHub/raul-portfolio/public/images/about/profile.webp
```

## Accepted Formats
- profile.webp (preferred)
- profile.jpg
- profile.png

## Verify
After saving, run:
```bash
npm run image:check
```
