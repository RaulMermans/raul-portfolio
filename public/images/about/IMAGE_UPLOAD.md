# Upload Your Portrait Image Here

## Quick Instructions

1. **Save your portrait image** in this folder (`/public/images/about/`)
2. **Name it**: `profile.webp` (or `profile.jpg` / `profile.png`)
3. **Recommended size**: 800x1067px or larger (portrait orientation)
4. **Format**: WebP preferred, but JPG/PNG also work

The code is already set up to use `/images/about/profile.webp`

If you use a different filename, let me know and I'll update the code!
