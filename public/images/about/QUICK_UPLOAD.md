# Quick Upload Instructions

## Upload Your Portrait Image

1. **Save your image file here** in this folder:
   ```
   /public/images/about/profile.webp
   ```

2. **Accepted formats**: `.webp`, `.jpg`, `.jpeg`, `.png`

3. **Recommended size**: 800x1067px (portrait orientation)

4. **That's it!** The website will automatically use it.

## Verify It Worked

Run this command in your terminal:
```bash
npm run image:check
```

Or:
```bash
node scripts/upload-image.js check
```
