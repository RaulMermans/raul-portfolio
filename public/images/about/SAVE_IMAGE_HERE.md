# ⚠️ IMPORTANT: Save Your Portrait Image Here

## Your Portrait Image Location

Save your portrait image file in **this exact folder** with **this exact name**:

```
📁 /public/images/about/
   └── 📄 profile.webp
```

## Quick Steps

1. **Take your portrait image file** (the selfie reflection image)
2. **Rename it to**: `profile.webp` (or `profile.jpg` / `profile.png`)
3. **Save it here**: `/public/images/about/profile.webp`
4. **Done!** The website will automatically use it.

## File Name Options

The code accepts any of these:
- `profile.webp` ✅ (best - smallest file size)
- `profile.jpg` ✅
- `profile.jpeg` ✅
- `profile.png` ✅

Just make sure it's named `profile` with one of those extensions.
