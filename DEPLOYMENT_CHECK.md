# Deployment Verification Checklist

## What Should Be Visible After Deployment

### 1. Contact Section (Homepage)
- [ ] Large "LET'S WORK" heading (h1, very large font)
- [ ] Subtitle: "Ready to bring your vision to life? Let's start a conversation."
- [ ] "Get in Touch" button (above the form)
- [ ] Contact form (below the button)

### 2. Homepage Scroll Snap (Desktop only)
- [ ] Sections snap into place when scrolling
- [ ] Smooth section transitions

### 3. Visuals Page
- [ ] Project cards aligned with left "VISUALS" text
- [ ] Proper spacing and layout

### 4. Case Studies
- [ ] Proper two-column layout
- [ ] Text doesn't overflow or get cut off
- [ ] Good spacing throughout

### 5. About Page
- [ ] Clean layout, no chopping
- [ ] Text flows properly
- [ ] Contact section under photo

## If Changes Not Visible:
1. Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
2. Clear browser cache
3. Try incognito/private window
4. Check Railway deployment logs for errors
5. Verify latest commit is deployed
